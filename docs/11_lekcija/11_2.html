11.2.Квиз - целобројно и реално дељење¶

Питање 1.¶

    Q-12: Дат је Пајтон израз.

    72 // 7

    Које је од наведених тврђења тачно? Изабери тачан одговор:

  • Ознаком // је представљена операција целобројног дељења бројева 72 и 7.
  • Тачно
  • Ознаком // је представљена операција реалног дељења бројева 72 и 7.
  • Нетачно
  • Ознака // не представља никакву операцију у Пајтон језику, а читав запис је неразумљив Пајтон окружењу.
  • Нетачно

Питање 2.¶

    Q-13: Дат је Пајтон израз.

    98 / 2

    Које је од наведених тврђења тачно? Изабери тачан одговор:

  • Ознаком / је представљена операција целобројног дељења бројева 98 и 2.
  • Нетачно
  • Ознаком / је представљена операција реалног дељења бројева 98 и 2.
  • Тачно

Питање 7.¶

    Q-15: Нека je вредност променљиве a природан број дељив бројем 7. Шта је од наведеног тачно?

    Изабери тачан одговор:

  • Вредност a % 10 је једнака 7.
  • Нетачно
  • Вредност a % 7 је једнака 0.
  • Тачно
  • Вредност a // 10 је једнака 7.
  • Нетачно
  • Вредност a // 7 је једнака 0.
  • Нетачно

Питање 8¶

Нека су a и b природни бројеви већи од 0, и нека је:

c = a % b.

  1. За свако a и b, ако је a дељиво са b, c ће бити једнако 0.

  2. За свако a и b, ако је a мање од b, c ће бити једнaко 0.

  3. За свако a и b, ако је b мање од a, c ће бити једнaко b.

  4. За свако a и b, ако је a мање од b, c ће бити једнaко a.

  5. За свако a и b, ако је b мање од a, c ће бити веће од 0.

  6. За свако a и b, ако је b дељиво са a, c ће бити једнaко 0.

(Одговор упиши навођењем редних бројева линија У растућем редоследу, нпр. 12345)

    Q-17: На који од наведених начина се може добити број који је једнак цифри јединица неког броја, на пример 811? Изабери тачан одговор:

  • 811 % 10
  • Тачно
  • 811 // 10
  • Нетачно
  • 811 - 10
  • Нетачно
  • 811 / 10
  • Нетачно
